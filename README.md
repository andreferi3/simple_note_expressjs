# simple_note_expressjs

## endPoint LIST

###### 1.GET
- `/category`
- `/category/:id` (category by id)
- `/notes`
- `/notes/:id'` (notes by id)
- `/note` (welcome to express)

###### 2.POST
- `/category`
- `/notes`

###### 3.PUT
- `/category`
- `/notes`

###### 4.DELETE
- `/category/:id'` (delete by id)
- `/notes/:id'` (delete by id)
